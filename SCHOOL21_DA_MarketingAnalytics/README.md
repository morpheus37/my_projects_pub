# Маркетинговая аналитика | Yandex DataLens

**Инструмент:** Yandex DataLens  
**Период:** Май 2026  
**Датасет:** Данные о привлечении пользователей банковского приложения

---

## 📊 Описание датасета

| Поле | Описание | Тип |
|------|----------|-----|
| user_id | ID клиента | Целое число |
| register_dt | Дата привлечения | Дата |
| cost_of_utilization | Стоимость привлечения | Дробное число |
| LTV | Доход от клиента за всё время | Дробное число |
| channel | Канал привлечения | Строка |
| is_churn | Флаг ушедшего клиента (0/1) | Целое число |

---

## 📈 Вычисляемые поля

| Поле | Формула | Описание |
|------|---------|----------|
| Regs | COUNTD([user_id]) | Количество регистраций |
| Cost | SUM([cost_of_utilization]) | Затраты на канал |
| Revenue | SUM([LTV]) | Выручка от канала |
| CAC | SUM([cost_of_utilization]) / COUNTD([user_id]) | Стоимость привлечения клиента |
| ROAS_% | SUM([LTV]) / SUM([cost_of_utilization]) * 100 | Возврат на рекламные вложения |
| Churn rate | SUM([is_churn]) / COUNTD([user_id]) | Процент ушедших клиентов |
| Avg LTV | AVG([LTV]) | Средний LTV на пользователя |
| Profit | SUM([LTV]) - SUM([cost_of_utilization]) | Прибыль канала |
| ROMI_% | (SUM([LTV]) - SUM([cost_of_utilization])) / SUM([cost_of_utilization]) * 100 | Окупаемость маркетинга |
