# Авиа-аналитика | Yandex DataLens

**Инструмент:** Yandex DataLens  
**Период:** Май 2026  
**Датасет:** 999 записей о перелётах (2012-2013)

---

## 📊 Описание датасета

| Поле | Описание |
|------|----------|
| NUM | Порядковый номер записи |
| DATE | Дата вылета |
| FLIGHT | Номер рейса |
| FROM/TO | Код аэропорта вылета/прилета |
| DIST | Расстояние в пути, км |
| AIRLINE | Код авиакомпании |
| SEAT | Место в самолете (W=окно, M=середина, A=проход) |
| TICKET | Тариф (эконом/бизнес) |
| REASON | Цель поездки (B=работа, L=отдых) |
| city_rus/city_eng | Город вылета |
| country_rus | Страна |

---

## 📈 Созданные чарты

| Чарт | Тип | Измерение | Метрика |
|------|-----|-----------|---------|
| `distance` | Индикатор | DIST | SUM |
| `seat` | Круговая диаграмма | SEAT | COUNT |
| `departure` | Столбчатая диаграмма | FROM | COUNT (сортировка по убыванию) |

## 🎯 Ключевые выводы

1. **Общая дистанция всех перелётов:** 170 442 км
2. **Предпочтения по местам:** Большинство пассажиров выбирают места у окна (W)
3. **Самый частый аэропорт вылета:** Тюмень (TJM) — 18 вылетов
4. **Основные направления:** Связаны с Москвой (VKO, DME, SVO)
