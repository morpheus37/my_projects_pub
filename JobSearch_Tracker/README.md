# Job Search Tracker | Личный проект

**Инструмент:** Jira (Kanban-доска)  
**Период:** Июнь 2026 — настоящее время  
**Статус:** Активное использование  
**Цель:** Систематизация поиска работы и отслеживание прогресса

---

## 📊 Описание проекта

Личный трекер для управления процессом поиска работы. Помогает структурировать задачи, отслеживать отклики, готовиться к собеседованиям и анализировать результаты.

---

## 🗂️ Структура Kanban-доски

| Колонка | Описание | Примеры задач |
|---------|----------|---------------|
| **BACKLOG** | Задачи на будущее | Повторить теорию по SCRUM/KANBAN, написать 3 людям в LinkedIn |
| **TO DO** | Запланировано на эту неделю | Подготовить рассказ о себе (2-3 минуты), изучить компанию |
| **IN PROGRESS** | В работе | Найти 20 вакансий, отсортировать по приоритету |
| **WAITING** | Ожидание ответа | Отправить отклики (Сбер, Первый Бит) |
| **DONE** | Выполнено | Сделать резюме, подготовить LinkedIn/HH |

---

## 📋 Категории задач

| Категория | Цвет | Что включает |
|-----------|------|--------------|
| ПОИСК ВАКАНСИЙ | Фиолетовый | Поиск, анализ, фильтрация вакансий |
| СОБЕСЕДОВАНИЯ | Розовый | Подготовка, изучение компании, follow-up |
| ОТКЛИКИ | Оранжевый | Отправка резюме, cover letters |
| ОФФЕР | Фиолетовый | Сравнение условий, принятие решения |
| ПОДГОТОВКА | Серый | Резюме, LinkedIn, нетворкинг |

## 🛠️ Навыки, продемонстрированные в проекте

- Управление личным проектом (self-management)
- Приоритизация задач (MoSCoW, Eisenhower Matrix)

## 💡 Как использую

1. **Ежедневно:** обновляю статусы задач, добавляю новые вакансии
2. **Еженедельно:** анализирую прогресс, пересматриваю приоритеты
3. **Перед собеседованием:** создаю эпик с чек-листом подготовки
